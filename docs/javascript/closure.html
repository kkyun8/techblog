# クロージャとは

Closure は関数系プログラミン（Scala、Haskell など）でよく使う物だ。 ただ Javascript だけ使う概念ではないので、理解しておく。

MDN Link https://developer.mozilla.org/en-US/docs/Web/JavaScript/Closures

# Lexical Scoping

クロージャを理解するためには、Javascript のレキシカルスコープ（スコープ）の理解が必要。

スコープは簡単にいうと変数の有効範囲のことをいう。

宣言した変数を JS のどこまで参照できるのか、理解するとクロージャの理解に役に立つ。
